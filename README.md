# noorulhuda.github.io
ഇസ്ലാമിക ജ്ഞാനപ്രസരണ രംഗത്തും സന്മാർഗ്ഗിക മൂല്യങ്ങൾ സമൂഹത്തിലേക്ക് പകർന്നുനൽകുന്നതിലും മുൻപന്തിയിൽ നിൽക്കുന്ന ഒരു ഉന്നത മതപഠന കേന്ദ്രമാണ് 'നൂറുൽ ഹുദാ ഇസ്ലാമിക് ദർസ്'. പാരമ്പര്യ ഇസ്ലാമിക വിദ്യാഭ്യാസ രീതികളെ മുറുകെപ്പിടിച്ചുകൊണ്ട്, ആത്മീയ ഉന്നതിയും ധാർമ്മികബോധവുമുള്ള ഒരു ഉത്തമ തലമുറയെ വാർത്തെടുക്കുക എന്നതാണ് ഈ ദർസിന്റെ പ്രധാന ലക്ഷ്യം.
